About the project

As cloud adoption continues to grow rapidly at the enterprise level, IT and security departments must provide for secure use of cloud storage and services. In computer science, Secure Data Storage refers to the store of data such as confidential or proprietary information over a secure channel. Many secure data storage over cloud methods require a type of encryption.

While cloud storage is convenient and gives employees access to their data anywhere, at any time, on nearly any device, cloud storage security is a top concern for organizations’ IT and security departments. The benefits brought by cloud storage – from scalability and accessibility to decreased IT overhead – are driving rapid adoption at enterprises around the world, and there are steps that companies should take to improve cloud storage security and keep sensitive data safe and secure in the cloud. We have developed SafeSync Storage- Encrypted Data Synchronization on Cloud web application using Python, Django and Mysql Technologies Using AES Encryption algorithm.
